# Waves & Wifi — Roamera Concept Listing Page

## Task
Build a single standalone `index.html` file that reimagines the Waves & Wifi coliving listing as a best-in-class Roamera property page. This is a design concept to show Tony what a captivating listing could look like.

## Design Direction
- Modern, warm, editorial feel — think Airbnb Luxe meets surf magazine
- Roamera brand colors: warm coral/terracotta (#E07A5F), cream (#FAF3E0), deep charcoal (#2D3436), sage green (#81B29A)
- Use the Roamera logo font vibe but this is standalone — no React, no build tools, just HTML + CSS + inline JS
- Mobile-responsive
- Images are in the `images/` folder — reference them as `images/filename.jpg`
- Heavy on visual storytelling — big photos, immersive sections
- Capture the "Surf, Yoga, Get Shit Done" energy
