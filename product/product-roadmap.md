# Roamera — Product Roadmap

## Sections (ordered by development priority)

### 1. Listing Page (Designed ✅)
The core page — a single coliving detail view. Concept: `index.html`
- Photo gallery, property header, highlights bar
- Section nav (scrollspy)
- About, Rooms, Coworking, Amenities, Community, Reviews, Location, FAQ
- Sticky booking card sidebar
- Intel/data overlay mode

### 2. Homepage (Designed ✅)
Discovery entry point. Concept: `homepage.html`
- Hero with search bar
- Featured cities carousel
- Featured/trending spaces
- How it works
- Community/social proof

### 3. Browse & Search (Partially designed)
Search results grid with filters.
- Card grid (3 columns desktop)
- Filter sidebar (city, price, amenities, stay length)
- Map view toggle
- Sort options

### 4. City Guide Pages (Not yet designed)
- `/city/:slug`
- City overview, coliving listings, neighborhoods, cost of living

### 5. Compare Tool (Not yet designed)
- Side-by-side comparison of 2-3 spaces

### 6. Reviews & Community (Partially designed in listing page)
- Standalone review feed page
- Write review flow
- Stay verification

### 7. Host Dashboard (Functional, needs design pass)
- Property management
- CRUD, photo management, publish/unpublish

### 8. Admin / Intel Dashboard (Designed ✅)
- Data quality view. Concept: `intel.html`
- Source attribution, completeness scores
