How It Works in Production

The template system automatically shows the appropriate CTA based on property data:

{{#if booking_url}}
  <a href="{{booking_url}}">Check Availability on Booking.com</a>
{{else if website}}
  <a href="{{website}}">🌐 Visit Website</a>
{{else if google_places_id}}
  <a href="https://maps.google.com/...">🗺️ View on Google Maps</a>
{{/if}}

All 89 generated listings use this conditional logic. When you sign up for affiliate programs, we'll update the booking_url field in the database.
